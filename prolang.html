Chapter 6: Programming Languages
TERM DEFINITION
machine language computer instructions encoded in numeric digits
debugging the process of correcting errors in a program
assembler a program that converts mnemonic expressions into machine language
assembly language a mnemonic system for representing programs
translator a program that translates a high-level program into a machine-language program
compiler a translation program that can compile machine instructions used to simulate a high-level primitive
interpreter a program that executes instructions as they are translated
cross-platform software software that can be executed independently from an operating system and hardware
natural language a language that develops naturally over time like the English language
formal language a language that is developed for a specific purpose such as programming
machine independence a program that can operate on many types of hardware
high-level programming language a formal language at a high level of abstraction from machine code that makes it easy for humans to program
object-oriented programming (OOP) a programming paradigm that views software as a collection of objects that are capable of performing actions for itself and other objects
scripting language a programming language that is used to perform administrative tasks or direct the execution of other programs
data types:
  • integer- numeric data stored as whole numbers
  • float - numeric data stored using floating-point notation
  • string - text data stored as characters
primitive data type a built-in data type for a programing language
data structure how data is arranged in a program
Python collection data structures:
  • list - changeable, ordered items stored in one variable using square brackets that can have the same value
  • tuple - unchangeable, ordered items stored in one variable using round brackets that can have the same value
  • set - unordered, unchangeable items stored in one variable using curly brackets that cannot be the same value
  • dictionary - ordered, changeable items stored in one variable using curly brackets and key: value pairs that do not allow duplicate keys
function a set of instructions used to perform a task
local variable a variable declared inside a function that can only be used in that function
global variable a variable declared outside a function that can be used by any function
source program a program's original form
object program a program's translation
