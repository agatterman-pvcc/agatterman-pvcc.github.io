Chapter 4 Networking & The Internet
TERM DEFINITION
Network Classifications
  • LAN – a network of computers in a small, limited area
  • MAN – an intermediate-size network for a metropolitan area
  • WAN – a large network spanning a geographical distance
Network models
  • bus network – connects all machines on a network through a bus
  • star network – uses a central focal point to connect machines
Protocols
  • CSMA/CD Ethernet – shares messages with all machines on the bus when the bus is not in use
  • CSMA/CA Wifi – forces new messages to wait a short period of time before sending
Combining networks:
  • repeater – passes signals back and forth between two buses
  • bridge – forwards a signal between two buses when the destination address is on the other bus
  • switch – a bridge that can serve more than two buses
  • router – connects networks together to form an internet
  • gateway – the point where a network is linked to an internet
Client/server model
  • P2P: peer-to-peer – machines can function as both servers and clients
  • distributed system – software components that are shared on different computers that are linked together using a network
  • cloud computing – the network of remote computers on the Internet that can be allocated for clients as needed
Internet Addressing
  • Tier 1 – high-speed, high-capacity, international WANs that form the backbone of the Internet
  • Tier 2 – regional ISPs that connect Tier 1 and Tier 3 providers
  • Tier 3 – service providers for end users that charge for Internet access
IP address
  • ICANN – a non-profit corporation established to coordinate the Internet's operation
  • DNS – an Internet-wide directory
  • TLD: Top-level domain – suffixes at the end of a domain name that are used to reflect the domain's classification
  • Country-code TLDs – two-letter domain name suffixes used to indicate a domain's country
Internet protocols
  • FTP – allows access to another computer from a great distance
  • HTTP – allows users to communicate data on the Internet
  • SMTP – defines how two computers on a network interact when transmitting an email
World Wide Web
  • URL – a unique address for a document on the Internet
  • client-side activities – operations performed by the client in an Internet browser
  • server-side activities – operations performed by a web server
Internet four-layer software
  • application layer – prepares messages, provides destination addresses, and receives messages
  • transport layer – chops messages into packets and collects packets for reassembly
  • network layer – assigns intermediate addresses to each packet and detects that the packets have reached their final destination
  • link layer – transfers and receives packets
Internet is:
  • a packet-switching network – that sends digital data by breaking it into packets, which contain data and control information for routing data
  • fault tolerant – so that it can continue operating without interruption when one or more components fail
  • redundant – in order to increase reliability and scalability by providing more than one way to route data
Cybersecurity
  • malware – malicious software
  • virus – software that infects a computer by inserting itself into programs within the machine
  • worm – an autonomous program that transfers itself through a network
  • Trojan horse – a program that enters a computer system disguised as a desirable program
  • spyware – software that collects information on the computer and reports back to the instigator of the attack
  • phishing – a technique of obtaining information explicitly by asking for it
  • DDoS attack – the process of overloading a computer with a large number of messages in a short period of time
Protection
  • firewall – a program that filters traffic passing through a point in the network to protect an individual computer
  • spam filter – a firewall designed to block unwanted email
  • proxy server – software that acts as an intermediary between a client and server that shields the client from adverse actions from the server
  • antivirus software – software used to detect and remove the presence of known viruses and infections
  • cryptography – secure communication techniques that use encryption to encode information
  • public key encryption – a system that uses public and private keys to secure messages using open standards
