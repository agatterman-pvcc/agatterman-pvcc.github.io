Operating Systems Terminology
TERM DEFINITION
operating system the software that controls the overall operation of a computer
job queue storage organization in which objects, or jobs, are ordered first-in, first out
FIFO first-in, first-out
interactive processing additional memory space made by using a technique called paging that rotates programs
and data back and forth between main memory and mass storage
time-sharing the ability of an operating system to service multiple users at the same time
multiprogramming a technique in operating systems that divides time into intervals and restricts each job
to use one interval at a time
application software programs used for performing tasks particular to the machine's utilization
system software programs used for performing tasks that are common to computer systems in general
utility software a category of system software consisting of programs that perform activities
fundamental to the computer's installation
graphical user interface (GUI) a user interface that represents objects pictorially on a display
shell a user interface that communicates with users through textual messages on a display
kernel the internal part of an operating system that contains software components used
for performing the very basic functions required by the computer installation
device driver a software unit that communicates with controllers or peripheral devices to carry out
operations on the peripheral devices attached to the machine
file manager a software component of an operating system's kernel that coordinates the use of
the machine's mass storage facilities
memory manager a software component of an operating system's kernel that coordinates the machine's
use of main memory
virtual memory additional memory space made by using a technique called paging that rotates programs
and data back and forth between main memory and mass storage
boot strapping a procedure that transfers the operating system from mass storage into main memory
when a computer is booted
ROM a small portion of a computer's main memory constructed from special nonvolatile
memory cells that are rarely altered
deadlock a condition in which two or more processes are blocked from progressing because each
is waiting for a resource that is allocated to the other
spooling the technique of holding data for output at a later, more convenient time
super user (or administrator) the highest privilege level for an operating system
multi-factor authentication an authentication method that requires multiple factors to gain access to an account
